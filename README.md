# Macroeconomic Analysis System - V1

## Project Overview

This project analyzes Costa Rica's GDP growth using real data from the World Bank.

## Features

- Data extraction from CSV (World Bank)
- Filtering Costa Rica data
- Statistical analysis (mean, max, min)
- Data visualization using Matplotlib
- Project structure for data analysis workflow

## Technologies Used

- Python
- pandas
- matplotlib

## Output

The system generates a chart showing GDP evolution:

`Outputs/Charts/pib_costa_rica.png`
